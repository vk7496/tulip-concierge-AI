# Tulip Concierge AI

Tulip Concierge AI is a bilingual (Arabic & English) digital concierge designed for Royal Tulip Muscat.  
It helps guests access hotel services, get instant answers to common questions, and request assistance, while reducing front desk workload and improving overall guest experience.

---

## Key Features
- Bilingual support (Arabic & English)
- AI-powered hotel concierge
- Quick access to hotel services and FAQs
- Guest-friendly chat interface
- Management insight dashboard (demo)
- No PMS integration required

---

## Use Cases
- Reduce front desk calls
- Improve guest satisfaction
- Provide 24/7 digital assistance
- Support both local and international guests

---

## Tech Stack
- Python
- Streamlit
- Groq API (LLM)
- Pandas
